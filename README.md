# HireMatch Landing

Landing page de lanzamiento del proyecto **Hire Match**, realizada como parte de la asignatura **DESSI** del Máster en Ingeniería Informática (curso **2025–2026**).

## Objetivo
Servir como una página sencilla de presentación/“coming soon” para comunicar la propuesta de valor de Hire Match y centralizar el primer punto de entrada al proyecto.

## Tecnologías
- HTML
- CSS
- JavaScript (Vanilla)

## Estructura del proyecto
Proyecto intencionalmente minimalista, compuesto por tres archivos en la raíz del repositorio:

- `index.html` – estructura y contenido
- `styles.css` – estilos
- `script.js` – comportamiento e interacción
